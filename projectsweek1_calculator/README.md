# Command Line Calculator

This calculator simulates a command line calculator. It can perform operations such as Addition, Subtraction, Multiplication, Division, and Modulus. 

The special part of this program is it's error handling for division by zero and for invalid inputs when requesting numbers. The program keeps running until you type 'quit' in the while loop.

## How to Use

To use this calculator, open `calculator.py` with Python, enter the desired operator, and input the numbers (a and b) for the calculation.
